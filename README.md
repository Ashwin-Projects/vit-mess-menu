# vit-mess-menu

VIT MessIt is a simple mess menu website made for students to check daily and weekly food menus. It has separate mess options like Veg, Non-Veg, and Special, and the menu changes based on the selected preference.

## Features
- Login and register page
- User details stored using browser localStorage
- Dashboard with today’s summary
- Veg, Non-Veg, and Special mess switching
- Today’s menu page
- Weekly schedule page
- Favorite food page
- Star button to add food items to favorites
- Dark mode and light mode toggle
- Favorites are saved separately for each logged-in user

## Pages
- `login.html` - Login and register page
- `index.html` - Main dashboard
- `home.html` - Today’s menu
- `weekly.html` - Weekly schedule
- `favorites.html` - Favorite food items

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap
- LocalStorage

## LocalStorage Usage
This project uses localStorage to save:
- Registered users
- Login session
- Selected mess type
- Favorite food items
- Dark/light theme preference

This is only for a basic frontend project. It is not meant for real secure authentication.

## Project Purpose
This project was made as a student web development project to make mess menu checking easier and more organized.
